# tictactoe

Basic Tic-tac-toe (noughts and crosses) implementation using CSS grid and Web Components.

## Demo

https://dutchcelt.github.io/tictactoe/

### Dependencies

-   NPM / Node
-   Yarn

### Install

`npm i`

### Script Options

Run the game, run tests or set development mode

-   npm test
-   npm run develop
-   npm run game
