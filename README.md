# tictactoe

Basic Tic-tac-toe (noughts and crosses) implementation using CSS grid and Web Components.

## Demo

https://dutchcelt.github.io/tictactoe/

## Dependencies

-   NPM / Node (v20)

### Install

`npm ci`

### NPM Script Options

Run the game, run tests or run in development mode

-   `npm run game`
-   `npm test`
-   `npm run develop`
